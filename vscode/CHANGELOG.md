# Changelog

All notable changes to the "fidan" extension will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
Fidan uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html) for
compiler releases.

---

## [Unreleased]

---

## [v1.0.8] - 2026-04-04

- Update `TextMate` grammar to support the latest syntax changes in Fidan

---

## [v1.0.7] - 2026-04-04

- Update extension to ensure compatibility with the latest version of the Fidan CLI

---

## [v1.0.6] - 2026-04-04

- Update VS Code explain commands to use the renamed CLI:
  - `fidan explain --diagnostic`
  - `fidan explain`

---

## [v1.0.5] - 2026-04-04

- Minor updates to VS Code commands

---

## [v1.0.4] - 2026-04-04

- Add first-class `WeakShared(...)` constructor highlighting
- Add `Shared` / `WeakShared` snippets

---

## [v1.0.3] - 2026-04-04

- Fix `powershell` command execution on Windows
- Fix highlighting issues in `fidan` files

---

## [v1.0.2] - 2026-04-03

- Fix extension activation issues

---

## [v1.0.1] - 2026-04-03

- Reduce extension size by removing unnecessary files
- Update [README](./README.md) with installation instructions

---

## [v1.0.0] - 2026-04-03

- Initial release
